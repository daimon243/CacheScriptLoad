Class: CacheScriptLoad

CacheScriptLoad

Part of the software package "Ws2Pg" for working with PostgreSQL over WebSocket. Simple localStorage based caching of JavaScript and css files. Cache variants see cacheUseVariant. Support script load order after loaded other scripts if need see cfg. After loading scripts may be run callback function see cfg. thanks to the author ideas: https://github.com/webpgr/cached-webpgr.js http://webpgr.com by Falko Krause
